# MicroML AST WebApp

This project is a simple single-page web application built with .NET.  
It allows users to input MicroML code and see the generated Abstract Syntax Tree (AST).

## Features
- Parse MicroML code
- Display AST tree visually
- Styled with CSS
- Single Page Application

## Technologies
- ASP.NET Core
- C#
- HTML/CSS
